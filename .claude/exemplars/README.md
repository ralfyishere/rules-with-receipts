# Exemplars

Real, rubric-graded PASS outputs from the pack's eval suite — kept verbatim (harness headers and workspace diffs included) so nothing is idealized. Use them as the reference standard: when unsure what "good" looks like for a debugging writeup, a scoped diff, a decision memo, or synthesis of delegated work, read the matching exemplar and match its shape and rigor, not just its rules.

Provenance: eval-results-v2/ (2026-07-07), conditions and rubrics in that folder. When a better graded artifact appears in a future eval, replace the exemplar and note it in CHANGELOG.md. Keep this folder small — 4-6 exemplars that get read beat 20 that don't.
